# Evidence Rubric

Score each criterion from 0–3 only when the required artifact exists.

| Criterion | 0 | 1 | 2 | 3 |
| --- | --- | --- | --- | --- |
| Product hierarchy | No discernible priority | Labels exist but regions compete | Primary action and regions are clear | Hierarchy remains clear across states and sizes |
| Adaptive composition | Fixed or broken | Scales without meaningful reflow | Purposeful compact and expanded layouts | Shared components preserve hierarchy across constraints |
| State completeness | Populated only | One additional state | All requested states exist | Local/global ownership and recovery are clear |
| Accessibility | No evidence | Basic semantic controls | Semantics, focus, scaling, reduced motion addressed | Automated evidence plus documented manual limits |
| Visual system | Scattered styling | Repeated constants | Semantic roles and coherent components | Domain distinction within one resilient system |
| Verification | No checks | Analyzer or tests only | Analyzer, widgets, and screenshots | Deterministic matrix inspected and refined |
| Behavior preservation | Contracts changed | Preservation unclear | Models and callbacks preserved | Preservation demonstrated by focused tests |

## Reporting

List artifact paths beside every score. Use `unverified` instead of a numeric score when evidence is missing. Report analyzer/test failures separately so a visual score cannot conceal correctness problems.
