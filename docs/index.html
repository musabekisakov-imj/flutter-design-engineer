SPECIALIST ROSTER / 04

Seven skills. Clear responsibilities.

Use the orchestrator for complete product work or invoke one specialist directly.

01 / ORCHESTRATOR

flutter-design

Routes and gates the full design workflow from repository inspection through rendered verification.

02 / DIAGNOSE

flutter-audit

Finds hierarchy, coherence, interaction, and adaptive-layout problems without unauthorized edits.

03 / SYSTEM

flutter-design-system

Builds semantic tokens, themes, typography, spacing, and reusable component contracts.

04 / BUILD

flutter-implementation

Implements approved production-quality interfaces with complete states and adaptive behavior.

05 / MOTION

flutter-motion

Adds purposeful, interruptible, performant motion with reduced-motion support.

06 / INCLUSION

flutter-accessibility

Hardens semantics, focus, text scaling, localization safety, and RTL behavior.

07 / EVIDENCE

flutter-visual-qa

Verifies screenshots, breakpoints, themes, state matrices, and golden-test evidence.

MAINTAIN + VERIFY / 05

Stay current without losing control.

The installer protects existing directories. Use --force only when replacement is intentional.

Install selected skills
python3 scripts/install.py --destination ~/.codex/skills --skill flutter-design --skill flutter-visual-qa
Update intentionally
python3 scripts/install.py --destination ~/.codex/skills --force
Verify seven directories
find ~/.codex/skills -maxdepth 1 -type d -name 'flutter-*' | sort

TROUBLESHOOTING / 06

Known problems. Direct answers.

npx is unavailable

Install Node.js, or use the HTTPS + Python direct command in the agent command center.

The destination already exists

Review the installed files first. Add --force only when you intend to replace them.

The agent cannot see the skills

Confirm the destination and fully start a new agent session after installation.

GitHub reports an SSH error

Use the HTTPS commands on this page. They do not require a GitHub SSH key.
