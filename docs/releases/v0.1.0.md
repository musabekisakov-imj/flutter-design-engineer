# Flutter Design Engineer v0.1.0

The first public release introduces a modular, evidence-oriented design-engineering workflow for Flutter coding agents.

## Included

- Seven self-contained skills for orchestration, audit, design systems, implementation, motion, accessibility, and visual QA.
- Progressive references covering anti-slop judgment, adaptive layouts, states, navigation, architecture, localization, and rendered QA.
- A safe local installer and deterministic repository validator.
- Seven behavioral eval cases with a shared scoring rubric.
- Compile-verified Flutter starter assets.
- A real adaptive booking demo with deterministic before/after golden screenshots.
- GitHub Actions, contributor templates, roadmap, security policy, and community entry points.

## Verification

- Repository validator passes.
- Python unit tests pass.
- Every skill passes the official skill validator.
- Flutter demo passes `flutter analyze` and widget/golden tests.

The project is new and seeks honest real-world feedback. No adoption or contributor thresholds are claimed in this release.
